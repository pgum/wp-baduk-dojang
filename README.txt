Use Baduk Dojang as League/School management with groups of players playing with each other for points.


== Description ==

Baduk Dojang is a project to manage school league, where players are playing against each other in league.
This plugin is primary for game of Go (or Baduk - games Korean name ).

League contains player groups, players should be divided by strength.


== Installation ==

Download as zip. Install from Plugins>Add Plugin.
